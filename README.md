# Prosthetic Outreach Coordination

Volunteer task coordination for our prosthetic outreach initiative, focused on identifying and connecting with clinical trials for affordable prosthetic solutions.

## 🎯 Mission

Coordinate volunteer teams across three regional hubs — **Chicago, Boston, and Atlanta** — to:
- Identify relevant clinical trials for affordable prosthetics
- Plan and execute outreach logistics
- Share resources and best practices across teams

## 📂 Project Structure

| Directory | Purpose |
|-----------|---------|
| `research/` | Clinical trial data, summaries, and analysis |
| `logistics/` | Outreach planning, travel, and venue coordination |
| `outreach/` | Community engagement templates and materials |
| `finance/` | Budget tracking and funding resources |

## ✅ How to Contribute

1. Check the **Issues** tab for active tasks — look for `good first issue` labels.
2. Claim a task by adding your username as a comment.
3. Use **GitHub Projects** for visual task board tracking.
4. Join **Discussions** for team communication and questions.
